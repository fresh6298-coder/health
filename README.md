# 🏋️ 건강관리 · 운동 & 인바디

점심시간 40분 운동 루틴을 **주간 체크리스트**로 관리하고, **인바디 기록**을 쌓아
체중·체지방률·골격근량 추이를 목표치와 비교해 볼 수 있는 웹앱입니다.

서버가 필요 없는 **단일 HTML 파일**이라, 파일 하나만 열면 바로 사용할 수 있어요.

## 주요 기능

- **📍 오늘** — 오늘 요일에 맞는 운동을 바로 보여주고, 이번주 진행률과 최근 4주 완료율,
  최신 인바디 vs 목표 진행 상황을 한눈에 확인
- **🗓️ 주간운동** — 요일별 운동(월 상체·화 유산소+코어·수 휴식·목 하체·금 전신·주말 보충)을
  실제 날짜 기준으로 체크. 이전주/다음주 이동 가능, 완료한 날은 초록 점으로 표시
  - 세트/횟수/휴식 시간, 자세 큐(cues), 타임라인까지 상세 표시
- **📈 인바디 기록**
  - 수동 입력 폼 또는 **InBody 기기에서 내보낸 CSV 파일을 그대로 업로드**하면 자동 인식
  - 체중 · 체지방률 · 골격근량 추이를 그래프로 표시 (목표선 포함, 호버 시 정확한 값 툴팁)
  - 전체 기록 테이블, 개별 기록 삭제
- **🎯 목표 관리** — 목표 체중 / 체지방률 / 골격근량 / 주간 목표 세션 수를 직접 설정
- **📲 앱 설치(PWA) + 오프라인** — 홈 화면/바탕화면에 설치, 오프라인에서도 사용 가능
- **💾 백업(내보내기/가져오기)** — 운동 기록·인바디 기록·목표를 파일로 저장하고 복원
- **🌓 다크/라이트 테마** — 시스템 설정 자동 감지 + 수동 전환

모든 기록은 **내 브라우저에만** 저장됩니다(localStorage). 외부 서버로 전송되는 데이터는 없습니다.

## 사용 방법

### 1. 앱 열기
`index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)로 열면 됩니다.

### 2. 오늘 운동하기
**📍 오늘** 탭에서 오늘 요일에 맞는 운동을 확인하고, 각 종목의 `완료 체크`를 눌러 기록하세요.

### 3. 인바디 기록 추가하기
**📈 인바디** 탭에서:
- InBody 기기 CSV를 그대로 업로드하거나
- 체중/골격근량/체지방률 등을 직접 입력

측정할 때마다 추가하면 그래프에 자동으로 반영됩니다.

### 4. 목표 확인/수정
**⚙️ 설정** 탭에서 목표 체중·체지방률·골격근량을 조정할 수 있어요.
기본값은 InBody 리포트의 체중조절 권장치를 기준으로 설정되어 있습니다.

### 5. 앱 설치
지원 브라우저에서는 설정 탭에 `📲 설치하기` 버튼이 나타납니다.
설치하면 앱처럼 실행되고, 인터넷이 없어도 화면과 기록을 볼 수 있습니다.

### 6. 백업
설정 → 백업에서 `내보내기`로 파일 저장, `가져오기`로 복원합니다.

## 온라인에 올리기 (선택)

GitHub Pages로 무료 배포할 수 있습니다.

1. 이 저장소의 **Settings → Pages**
2. Source를 배포 브랜치로 지정
3. 발급된 주소로 접속 (예: `https://<사용자명>.github.io/health/`)

> **PWA(설치·오프라인)는 `https`로 호스팅해야 동작합니다** (예: GitHub Pages).
> 파일을 직접 열어도(`file://`) 앱은 잘 동작하지만, 설치/오프라인 기능만 비활성화됩니다.

## 폴더 구성

```
index.html               # 앱 전체 (단일 파일)
manifest.json            # PWA 설정 (설치 정보)
sw.js                    # Service Worker (오프라인 캐시)
icon.svg                 # 앱 아이콘
icon-maskable.svg        # 안드로이드 마스커블 아이콘
```

## 기술 메모

- 단일 `index.html` (외부 라이브러리 없음)
- 저장: `localStorage` (운동 체크·인바디 기록·목표), 백업은 JSON 파일
- CSV 파싱: InBody 기기 표준 내보내기 포맷(헤더 기반 매핑, `-`는 결측치로 처리)
- 그래프: 순수 SVG (외부 차트 라이브러리 없음), 호버 툴팁 포함
- 설치/오프라인: Web App Manifest + Service Worker (PWA)
- 다크 모드 자동 지원 + 수동 전환
